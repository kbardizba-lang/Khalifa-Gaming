# Khalifa Gaming

Website hii inatoa njia rahisi ya **kupakua games** kwa haraka na salama.

## Maelezo ya Project

- Inajumuisha: `index.html`, `style.css`, na `game.js`
- Inafanyaje kazi: Inakuwezesha kuona games na kupakua
- Lengo: Kuboresha gaming experience kwa Watanzania na watu wote

## Jinsi ya Kutumia

1. Fungua `index.html` kwenye browser yako (Chrome, Firefox, Edge)
2. Bonyeza game unayopenda au bidhaa (bidhaa)
3. Furahia kupakua na kucheza!
